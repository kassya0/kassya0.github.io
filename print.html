3D-Model

Using the sculpture we've created in Minecraft, we used a 3D model for to print it out. On the left is the first draft of the model print. Some changes were made compared to the initial design of my Pacman sculpture. Initially, I was encouraged to build up on the sculpture and avoid using any 2D elements, as the goal was to create something that can be fully experienced in 3D. So, I built upon the original concept, which led to the first draft. However, I wasn't satisfied with the result, so I decided to go back and make further adjustments to my Pacman sculpture. After reworking it, I printed it again, and the final result is shown on the right.
